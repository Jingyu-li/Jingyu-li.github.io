本文详细记录了利用Hexo+Github搭建个人博客的完整过程。Hexo是一个快速、简洁且高效的博客框架，可以快速生成静态网页，并托管在github等平台上，同时对markdown有很好的支持。

相关工具和技术

  • github
  • node.js/npm (node.js是一个JavaScript的运行环境，npm是随同node.js一起安装的包管理工具)
  • hexo
  • markdown
